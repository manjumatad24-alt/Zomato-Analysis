# Zomato Dataset Analysis

## Dataset
Kaggle - Zomato Restaurant Dataset
Link: https://www.kaggle.com/datasets/bhanupratapbiswas/zomato

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, WordCloud

## Key Findings
- Total restaurants analysed: 30,864
- Average rating: 3.71 / 5
- Average cost for two: ₹614
- Most popular cuisine: North Indian
- Most popular location: BTM
- Restaurants with online ordering: 20,217
- Restaurants without online ordering: 10,647

## Visualizations
- Top 10 Cuisines by Restaurant Count
- Top 10 Restaurant Locations
- Price vs Rating Scatter Plot
- Online Order Availability Pie Chart
- Distribution of Restaurant Ratings
- Cuisine Word Cloud

## Business Recommendations
1. Partnership focus on top locations (BTM, Koramangala)
2. Cuisine content strategy for North Indian and Chinese
3. Online ordering push for 10,647 restaurants
4. Rating improvement program for low rated restaurants
5. Value picks feature for mid-range restaurants

## Deliverables
- Jupyter Notebook with cleaned data and visualizations
- PDF report with key findings and recommendations
